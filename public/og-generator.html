Cómo generar la imagen

  1. Abre esta página en Chrome o Firefox
  2. Presiona F12 para abrir DevTools
  3. Click derecho en el preview → "Capture node screenshot"
  4. Guarda como og-image.png en /public

O usa extensiones como "Full Page Screen Capture" o captura la pantalla (1200x630px)
